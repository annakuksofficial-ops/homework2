# Обработчик банковских операций

Python-модуль для фильтрации и сортировки банковских операций.

## Описание

Модуль предоставляет две основные функции для работы со списками банковских операций:

1. Фильтрация по статусу
```python
# Входные данные:
operations = [
    {'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'},
    {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'},
    {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'},
    {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}
]

# Фильтрация по умолчанию (статус 'EXECUTED'):
result = filter_by_state(operations, 'EXECUTED')
# Результат:
# [
#     {'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'},
#     {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}
# ]

# Фильтрация по статусу 'CANCELED':
result = filter_by_state(operations, 'CANCELED')
# Результат:
# [
#     {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'},
#     {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}
# ]
```
