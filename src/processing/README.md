# Обработчик банковских операций

Python-модуль для фильтрации и сортировки банковских операций.

## Описание

Модуль предоставляет две основные функции для работы со списками банковских операций:

1. Фильтрация по статусу
2. Сортировка по дате

## Установка

Скопируйте файл с функциями в свой проект.
